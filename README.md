# 🏗 Telegram-бот для обліку робочого часу (Time Tracker)

Автономний Telegram-бот для будівельної компанії, призначений для відстеження присутності працівників на різних об'єктах за допомогою GPS-геолокації. Бот мінімізує людський фактор, має автоматичні фонові задачі та зручну панель адміністратора.

## ✨ Основний функціонал

### 👷 Для працівників:
* **Проста реєстрація:** Одноразове введення ПІБ при першому запуску.
* **Відмітка в один клік:** Єдина кнопка "📍 Я на объекте". Бот сам визначає найближчий будівельний майданчик за GPS та перевіряє радіус допуску (до 150 метрів).
* **Захист від дублів:** Неможливо відмітитися двічі за один день.
* **Особистий кабінет:** Перегляд кількості відпрацьованих днів у поточному звітному періоді (1-15 або 16-кінець місяця).

### 👑 Для адміністратора:
* **Адмін-панель:** Викликається командою `/admin` або кнопкою. Доступна лише для вказаних Telegram ID.
* **Списки активних:** Можливість в один клік побачити, хто саме зараз знаходиться на об'єктах із розбивкою по локаціях (Кальдера, Нірвана, Офіс).
* **Масова розсилка:** Відправка важливих оголошень усім зареєстрованим працівникам прямо через бота (з вбудованим антиспам-захистом).
* **Загальна статистика:** Таблиця відпрацьованих днів по всій команді з можливістю динамічного оновлення повідомлення (кнопка "🔄 Обновить").

### 🤖 Автономні фонові задачі:
* **08:30 — Ранковий звіт:** Автоматична відправка адміну списку працівників, які вже вийшли на роботу, згрупованого по об'єктах.
* **16:00 — Авто-закриття зміни:** Бот автоматично проставляє час виходу (16:00) всім працівникам, які відмітилися зранку. Це повністю усуває проблему "забув натиснути кнопку виходу".

## 🛠 Технології та бібліотеки
* **Python 3.9+**
* **Aiogram 3.x** — асинхронний фреймворк для Telegram Bot API.
* **SQLite3** — вбудована легка база даних для збереження користувачів та відміток.
* **Geopy** — бібліотека для розрахунку відстані між GPS-координатами.
* **Python-dotenv** — для безпечного зберігання токенів у змінних середовища.

## 🚀 Встановлення та запуск

### 1. Клонування або завантаження проєкту
Переконайтеся, що всі файли (`main.py`, `Registration.py` тощо) знаходяться в одній папці.

### 2. Встановлення залежностей
Відкрийте термінал у папці з проєктом та встановіть необхідні бібліотеки:
```bash
pip install aiogram geopy python-dotenv
